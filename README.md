Telegram Bot 🤖

I made this bot for my sister, but you can use it too!

A functional Telegram bot built with Python that handles user registration and validates personal data.

Features
1. Age Validation: Processes user date of birth inputs and calculates/validates age using the datetime library.
2. Smart Chat Management: Automatically deletes service messages and error notifications to maintain a clean and professional user interface.
3. Intuitive Navigation: Features custom Inline Keyboards for smooth transitions between menus, including "Back" and "Main Menu" functionality.
4. Error Handling: Robust logic to guide users through correct data entry without cluttering the chat history.

Tech Stack
1. Language: Python 3.11
2. API Wrapper: [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
3. Deployment: Hosted on Amvera Cloud

Installation & Setup
1. Clone the repository.
2. Install dependencies: pip install -r requirements.txt.
3. Configure your BOT_TOKEN in the main script.
4. Run BotForMargo.py.
